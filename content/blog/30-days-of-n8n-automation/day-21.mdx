## **Engine 1: The "Content Refinery" (Video to Semantic Blog)**

The core philosophy of my automation strategy is **"Write Once, Distribute Everywhere."** But we don't just copy-paste; we transform.

We will build a workflow that takes a YouTube video (or an audio voice note) and transforms it into a high-quality, SEO-optimized blog post for your site.

### **Step 1: The Trigger & Transcription**

We start with the **YouTube Trigger** node. This polls your channel for new uploads.

- **Action:** When a new video is detected, we grab the `videoURL`.

- **The Technical Hurdle:** YouTube doesn't give you the file. We use an `Execute Command` node (if running via Docker) to run `yt-dlp` to extract the audio, or use an API like **Cobalt**.

- **Transcription:** We pass the audio file to the **OpenAI Whisper** node. This model is superior to standard speech-to-text because it handles technical jargon (like "n8n", "API", "JSON") with high accuracy.

### **Step 2: The Semantic "Agent Chain"**

This is where we differentiate from low-quality AI content. We don't just dump the transcript into ChatGPT. We use a **Chain of Thought** approach within n8n.

**Agent A: The Strategist**

- **Input:** The raw transcript.

- **System Prompt:** "You are an SEO Strategist. Analyze this transcript. Identify the Primary Keyword and 5 LSI (Latent Semantic Indexing) keywords. Output strictly in JSON format."

**Agent B: The Architect**

- **Input:** The transcript + The JSON keywords from Agent A.

- **System Prompt:** "Create a blog post outline. Ensure the H2 headers capture the LSI keywords. Structure the post for high readability (bullet points, short paragraphs)."

**Agent C: The Writer**

- **Input:** The Outline.

- **System Prompt:** "Write the full article in Markdown. Use a professional, authoritative tone similar to `whoisalfaz.me`. Avoid fluff words like 'delve', 'tapestry', or 'game-changer'."

### **Step 3: The Code Node (JSON Transformation)**

To make this data usable for WordPress, we need to merge these outputs.

JavaScript

```
// n8n Code Node: Preparing the WordPress Payload
const title = items&#91;0].json.title;
const content = items&#91;0].json.content; // Markdown from Agent C
const keywords = items&#91;0].json.keywords; // From Agent A

// Constructing the Slug
const slug = title.toLowerCase().replace(/ /g, '-').replace(/&#91;^\w-]+/g, '');

return {
  json: {
    title: title,
    content: content,
    slug: slug,
    status: 'draft', // Always draft for human review!
    tags: keywords
  }
};

```

### **Step 4: Publishing to WordPress**

We use the **WordPress Node** to create the post.

- **Crucial Step:** We map the "Primary Keyword" to the **RankMath** or **Yoast** focus keyword field (via custom meta fields) to ensure we are tracking our SEO performance immediately.

By running this workflow, you ensure that every video you create automatically contributes to your domain authority.

We will automate the **Technical Audit** using n8n and the **Google PageSpeed Insights API**.

### **Step 1: The Cron Scheduler**

We don't need this continuously. We set a **Cron Trigger** for **Every Monday at 03:00 AM**.

### **Step 2: The Audit**

- **Node:** **HTTP Request**.

- **URL:** `https://www.googleapis.com/pagespeedonline/v5/runPagespeed?url=https://whoisalfaz.me&strategy=mobile`

- **Method:** GET.
